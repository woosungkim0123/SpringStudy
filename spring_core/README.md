# Spring Core

## 소개

- 객체 지향 설계 원칙과 스프링의 핵심 기술에 대해 알아봅니다.
- OCP와 DIP를 지키면서 객체 지향적으로 설계하는 방법을 알아봅니다.
- 스프링 컨테이너의 역할을 이해하고 스프링 빈을 등록하는 방법을 알아봅니다.

- OCP, DIP 위반
- 좋은 객체 지향 설계의 원칙 적용
- IoC, DI, 그리고 컨테이너
